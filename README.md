# VibraSim Pro | Mechanical Vibrations Simulator

**VibraSim Pro** is an advanced, engineering-grade mechanical vibrations simulator built using **Streamlit**, **Numpy**, and **Plotly**. It provides an interactive environment to explore Single Degree of Freedom (SDOF) systems across various vibration regimes.

![App Header](assets/iconvibsem.png)

## 🚀 Features

-   **🎯 Multiple Vibration Modes**:
    -   Simple Harmonic Motion (SHM)
    -   Free Damped Vibrations (Underdamped, Critically Damped, Overdamped)
    -   Forced Damped Vibrations with Steady-State Analysis.
-   **📊 Advanced Analytics**:
    -   Time-domain plots for Displacement, Velocity, and Acceleration.
    -   2D and 3D Phase Space trajectories.
    -   Frequency Response curves (Magnification Factor & Transmissibility).
-   **🎥 Live Animation**: Real-time visualization of a mass-spring-damper system with a dynamic zigzag spring.
-   **📚 Theory Integration**: Built-in physics reference with LaTeX governing equations.
-   **📥 Data Export**: Download simulation results as CSV for external analysis.

## 🛠️ Installation

1.  **Clone the repository**:
    ```bash
    git clone <repository-url>
    cd <project-folder>
    ```

2.  **Install dependencies**:
    ```bash
    pip install streamlit numpy plotly pandas
    ```

## 🏃 Usage

Launch the simulator with the following command:
```bash
streamlit run app.py
```
